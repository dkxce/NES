Предварительно имеем распакованый Dendy_nestopia1.37.zip и распакованый Roms.rar
1. Настраиваем и запускаем сервер "kaillerasrv.exe"
1. Запускаем эмулятор (nestopia.exe)
2. Выбираем Netplay>Connect...
3. В появившемся окне кнопкой "add" добавляем ромы игр в которые планируеться играть по IP. (Ромы должны быть идентичными у обоих игроков!)
4. Нажимаем Launch...
5. В новом окне находим вкладку Recent, вводим "Username"
   находим или вводим наш сервер с помощью кнопки "Enter IP..."
   для локальной машины это может быть 127.0.0.1 или 127.0.0.1:27888 (с указанием порта),
   для удаленной машины IP адрес и порт соответственно
6. Нажимаем Connect - откроется чат
7. Чтобы подключиться к создаваемой игре, надо выбрать игру внизу из списка и нажать Join
   Чтобы создать игру, надо нажать Create New Game и выбрать игру из списка
8. После присоединения всех участников, тот кто создал игру кажимает Start Game
